## Описание программы

Random Quote Generator — это графическое приложение (GUI) для генерации случайных цитат. Программа позволяет добавлять записи о цитатах, включая текст, автора и тему. Приложение сохраняет историю всех сгенерированных цитат с указанием даты и времени. Пользователь может фильтровать историю по автору или теме, а также добавлять новые цитаты в общую коллекцию. Random Quote Generator решает проблему поиска вдохновения, помогает коллекционировать любимые высказывания и демонстрирует навыки работы с графическим интерфейсом, файловым вводом-выводом в формате JSON и валидацией пользовательского ввода.

---

## Требования для запуска

Для успешного запуска приложения на вашем компьютере должно быть установлено следующее: Python версии 3.6 или выше. Дополнительные библиотеки не требуются, так как используются только встроенные модули Python: tkinter (для графического интерфейса), json (для работы с файлами), random (для выбора случайной цитаты), datetime (для отметки времени в истории) и os (для проверки существования файлов). На некоторых операционных системах Linux может потребоваться отдельно установить пакет python3-tk командой sudo apt-get install python3-tk.

## Как запустить

Для запуска приложения откройте терминал или командную строку и выполните следующие команды. Сначала склонируйте репозиторий с проектом: git clone https://github.com/твой-логин/random-quote-generator.git. Затем перейдите в директорию проекта: cd random-quote-generator. После этого запустите главный файл приложения: python quote_generator.py. Если на вашем компьютере установлено несколько версий Python, используйте команду python3 quote_generator.py. После выполнения этих команд откроется графическое окно приложения, готовое к работе.

## Как пользоваться

После запуска приложения вы увидите главное окно с несколькими зонами. В верхней части отображается текущая сгенерированная цитата. Чуть ниже расположена зелёная кнопка «Сгенерировать цитату» — при её нажатии программа случайным образом выбирает одну цитату из общего списка, отображает её на экране и автоматически добавляет запись об этом в историю. В средней части окна находятся выпадающие списки для фильтрации истории по автору и по теме, а также кнопка «Сбросить фильтры». В нижней части окна расположен список истории, где каждая запись содержит дату и время генерации, автора, часть текста цитаты и тему. В самом низу окна находится кнопка «Добавить новую цитату» — при её нажатии открывается отдельное диалоговое окно с полями для ввода текста цитаты, автора и темы. После заполнения всех полей и нажатия кнопки «Сохранить» новая цитата добавляется в общий список и становится доступной для генерации. Все данные автоматически сохраняются в JSON-файлы при закрытии приложения.

## Пример работы

Рассмотрим несколько примеров использования приложения. При нажатии кнопки «Сгенерировать цитату» на экране появляется текст: «Будь изменением, которое ты хочешь увидеть в мире.» — Махатма Ганди (Тема: Мотивация). Одновременно с этим в истории появляется запись: [2026-05-15 14:32:10] Махатма Ганди: «Будь изменением, которое ты хочешь увидеть в мире...» (Тема: Мотивация). При добавлении новой цитаты пользователь заполняет три поля: текст цитаты «Программирование — это искусство объяснять компьютеру, что ты хочешь, на языке, который он понимает», автор «Алан Кей», тема «Программирование». После нажатия «Сохранить» программа сообщает об успешном добавлении, и новая цитата становится доступна для случайной генерации. При фильтрации истории по автору «Махатма Ганди» в списке истории остаются только цитаты этого автора, а все остальные временно скрываются. При нажатии кнопки «Сбросить фильтры» история снова показывает все записи без исключений.

## Архитектура проекта

Проект имеет следующую структуру. В корневой директории находится главный файл приложения quote_generator.py, который содержит весь код программы, включая класс QuoteApp и все вспомогательные функции. В процессе работы программа автоматически создаёт два JSON-файла: quotes.json для хранения всех цитат (как предопределённых, так и добавленных пользователем) и history.json для хранения истории сгенерированных цитат с отметками времени. Также в корне проекта располагаются файл README.md (этот документ) и файл .gitignore, который исключает из системы контроля версий временные файлы и кэш Python. Все данные сохраняются в текстовом формате JSON, что позволяет легко просматривать и редактировать их в любом текстовом редакторе.

## Тестирование

Приложение прошло тестирование на различных типах входных данных. Позитивный тест: нажатие кнопки «Сгенерировать цитату» при наличии цитат в списке приводит к появлению цитаты на экране и добавлению записи в историю — результат положительный. Негативный тест: попытка добавить новую цитату с пустым полем (например, оставить поле «Автор» незаполненным) приводит к появлению сообщения об ошибке «Все поля должны быть заполнены», при этом цитата не сохраняется — результат положительный. Граничный тест: добавление цитаты с очень длинным текстом (1000 символов и более) успешно сохраняется, история отображает сокращённую версию (первые 60 символов) без ошибок — результат положительный. Тест фильтрации: выбор в выпадающем списке автора, которого нет ни в одной записи истории, приводит к тому, что список истории становится пустым — результат положительный. Тест сохранения: закрытие приложения с последующим повторным запуском показывает, что все ранее добавленные цитаты и вся история загружаются из JSON-файлов в полном объёме — результат положительный.
